## 1. AMAZON ASSOCIATES (Prioridad Alta)

### Datos:
- URL registro: https://affiliate-program.amazon.com/
- Comision: 1% a 10% segun categoria
- Cookie: 24 horas (si agregan al carrito: 90 dias)
- Pago minimo: $10 (transferencia) o $100 (cheque)
- Frecuencia pago: Mensual (60 dias despues del mes)

### Comisiones por categoria:
- Moda/Ropa: 4-6%
- Electronica: 1-3%
- Belleza/Salud: 3-6%
- Hogar: 3-5%
- Alimentacion: 4.5%
- Lujo (Premium Beauty): hasta 10%

### Como registrarse:
1. Ir a https://affiliate-program.amazon.com/
2. Crear cuenta (o usar tu cuenta Amazon existente)
3. Agregar tu sitio web (LUXDEAL)
4. Completar perfil (nicho: comparador de precios/ofertas)
5. Elegir metodo de pago
6. Empezar a generar links

### Requisito importante:
- Necesitas hacer 3 ventas en los primeros 180 dias
- Tu web debe tener contenido real y trafico
- No puedes usar links de afiliado en emails directamente

### Formato de link:
```
https://www.amazon.es/dp/B0XXXXXXXXX?tag=tu-tag-21
```
